第1章 概要

TableEngine とは

TableEngine は、Windows 上で動作するオープンソースのファイルマネージャーです。 MS-DOS 時代のキーボード操作ファイラー FD、および Windows 向けの 卓駆☆ にインスパイアされ、これらが Windows 10/11 に対応していないことを きっかけに開発が始まりました。

従来のキーボード操作中心のファイラーの使い勝手を継承しつつ、 現代の Windows 環境で快適に動作することを目標としています。

主な特徴

  • デュアルペイン型のファイル管理（フォルダツリー + ファイルリスト）
  • タブ機能による複数フォルダの同時管理
  • ファイルの圧縮・展開（libarchive 対応）
  • ファイルの分割・結合
  • テキスト・バイナリビューア内蔵
  • ファイル名の一括リネーム
  • ファイル検索（名前・更新日・サイズ）
  • キーボードショートカットのカスタマイズ
  • メニュー構成のカスタマイズ
  • お気に入りフォルダ登録
  • 複数の選択スタイル（エクスプローラー風 / TableEngine 風）

動作環境

項目要件
OSWindows 10 / 11 (64bit)
QtQt 6.10 以降
コンパイラMicrosoft Visual C++ 2022
依存ライブラリlibarchive、libmagic、ICU libs
