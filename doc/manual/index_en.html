TableEngine User Manual

TableEngine is a keyboard-centric file manager for Windows. Inspired by the MS-DOS file manager FD and the Windows application Takuchu☆, it was developed to bring that style of keyboard-driven file management to modern Windows 10/11.

This software is currently under development. Some features may not be fully functional.
