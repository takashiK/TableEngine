Chapter 7: View Settings

You can finely control how the file list is displayed. These commands are accessible from the View menu. Settings work as toggles (ON/OFF).

Column Visibility

Show File Size

View › File Size
Toggles the file size column in the file list.

Show Modified Date

View › Modified Date
Toggles the last modified date column in the file list.

Hidden and System Files

Show Hidden Files

View › Hidden Files   H
Toggles the display of files and folders with the Hidden attribute. When shown, they appear with a dimmed icon.

Show System Files

View › System Files
Toggles the display of files and folders with the System attribute.

Accidentally modifying system files may destabilize the OS. Handle with care.

Filter

View › Filter
Narrows the file list display by file name pattern. Folders are not affected (unless the "Files Only" option is used).

OptionDescription
Filter StringWildcard or regex pattern to match file names
Files OnlyExclude folders from the results
Case SensitiveDistinguish upper and lower case when matching
Regular ExpressionTreat the input as a regular expression
The filter only affects what is displayed; it does not delete or move any files.
→ Dialog details: Filter Dialog

Sort

Select a sort criterion from the View › Sort submenu.

Sort ByDescription
NameAlphabetical order by file name
SizeBy file size
ExtensionAlphabetical order by extension
Modified DateBy last modified date

Ascending / Descending

View › Order › Ascending
View › Order › Descending
Choose the sort direction.

Layout

Select a display layout from the View › Layout submenu.

LayoutIcon SizeDescription
Small Icons 16×16 px (default) Small icons in a multi-column list
Large Icons 64×64 px Large icons in a grid layout
Huge Icons 192×192 px Huge icons in a grid layout.
Image files are displayed as thumbnails.
Detail List 16×16 px (default) Single-column list showing name, size, and modified date
