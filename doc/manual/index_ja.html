TableEngine 取り扱い説明書

TableEngine は Windows 上で動作する、キーボード操作を重視したファイルマネージャーです。 MS-DOS 時代の FD や Windows 向けの 卓駆☆ にインスパイアされ、 現代の Windows 10/11 環境に対応させることを目的として開発されています。

本ソフトウェアは現在開発中です。一部の機能は動作しない場合があります。
