Chapter 2: Installation & Launch

Required Files

The following files must be placed in the same folder as TableEngine.exe.

FileDescription
TableEngine.exeMain executable
archive.dlllibarchive (compression/extraction library)
libmagic.dllFile type detection library
magic.mgclibmagic database file
Qt DLLsQt6Core.dll, Qt6Gui.dll, Qt6Widgets.dll, etc.
Qt DLLs can be collected from your Qt installation folder or by running the windeployqt tool.

Installation

TableEngine has no installer. Follow these steps to set it up:

  1. Create a folder of your choice (e.g., C:\Tools\TableEngine\).
  2. Copy the built TableEngine.exe into that folder.
  3. Copy the dependency DLLs (archive.dll, libmagic.dll, magic.mgc) into the same folder.
  4. Copy the required Qt DLLs into the same folder.
    You can collect them automatically from a command prompt: 
    windeployqt.exe TableEngine.exe

Uninstallation

TableEngine does not use the Windows registry. Settings are saved in the user's application data folder.
To completely remove it, delete the installation folder and the settings folder below. 

%APPDATA%\TableEngine\

Launching

Double-click TableEngine.exe, or run it from the command line: 

TableEngine.exe
Pin it to the Start menu or taskbar for quick access.

Building from Source (Developers)

To build from source, follow these steps.

Prerequisites

  • Qt 6.10 or later (specified via Qt6_DIR or CMAKE_PREFIX_PATH)
  • CMake 3.19 or later
  • Microsoft Visual C++ 2022
  • Support libraries (libarchive, libmagic, valijson) placed under support_package/

Build Steps

cmake --preset "Visual Studio Community 2022 Release - amd64"
cmake --build out/build/"Visual Studio Community 2022 Release - amd64" --config Release

To open in Visual Studio:

cmake --preset "Visual Studio Community 2022 Release - amd64"
start out/build/"Visual Studio Community 2022 Release - amd64"/TableEngine.sln
