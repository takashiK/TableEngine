Chapter 2: Installation & Launch

Installation

TableEngine has no installer. Simply extract the distribution package (ZIP file) to complete the installation.

  1. Download the distribution package (e.g., TableEngine-0.1.0-win64.zip).
  2. Extract the ZIP file to a folder of your choice (e.g., C:\TableEngine).
  3. Once extracted, run TableEngine.exe from that folder to launch the application.

Uninstallation

TableEngine does not use the Windows registry. Settings are saved in the user's application data folder.
To completely remove it, delete the installation folder and the settings folder below. 

%APPDATA%\TableEngine\

Launching

Double-click TableEngine.exe, or run it from the command line: 

TableEngine.exe
Pin it to the Start menu or taskbar for quick access.

Building from Source (Developers)

To build from source, follow these steps.

Prerequisites

  • Qt 6.11 or later (MSVC 2022 64-bit kit; includes Ninja)
  • CMake 3.19 or later
  • Microsoft Visual C++ 2022
  • Git (required to fetch submodules)

Build Steps

1. Clone the repository and initialize submodules:

git clone <repo-url> TableEngine
cd TableEngine
git submodule update --init --recursive

2. Bootstrap vcpkg (first time only):

support\vcpkg\bootstrap-vcpkg.bat -disableMetrics

3. Configure and build using the CMake preset:

cmake --preset Qt-MSVC2022-amd64-Ninja
cmake --build out/build/Qt-MSVC2022-amd64-Ninja --config Release

4. Create the distribution package:

.\scripts\package-release.ps1
