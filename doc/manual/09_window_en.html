Chapter 9: Window

This chapter covers tab management and bar visibility. These commands are accessible from the Window menu.

Tab Operations

TableEngine can open multiple folders simultaneously using tabs. Click a tab in the tab bar to switch between them.

New Tab

Window › New Tab   Ctrl+N
Opens a new tab showing the current folder.

Close Tab

Window › Close Tab   Ctrl+W
Closes the current tab. The last remaining tab cannot be closed.

Move Tab to New Window

Window › Move Tab   Ctrl+T
Detaches the current tab into a new independent window. Useful for placing two windows side by side to copy files between them.

Bar Visibility

Each bar can be shown or hidden using the toggle (ON/OFF) in the menu.

CommandDescription
Window › Show Toolbar Toggle toolbar visibility
Window › Show Drive Bar Toggle drive bar visibility
Window › Show Status Bar Toggle status bar visibility
Window › Show Navigation Toggle the left-side folder tree (navigation pane)
Window › Show Details Toggle the right-side detail pane
Hiding the navigation pane and detail pane maximizes the file list area for a cleaner workspace.
