Chapter 5: Selection & Editing

This chapter covers file selection and clipboard operations. These commands are accessible from the Edit menu.

Clipboard Operations

Cut

Edit › Cut   Ctrl+X
Cuts the selected files/folders to the clipboard. They are removed from the original location when Paste is executed.

Copy

Edit › Copy   Ctrl+C
Copies the selected files/folders to the clipboard.

Paste

Edit › Paste   Ctrl+V
Pastes files/folders from the clipboard into the current folder. The clipboard is shared with Windows Explorer.

Selection Operations

Select All

Edit › Select All   A / Ctrl+A
Selects all files and folders in the current folder.

Invert Selection

Edit › Invert Selection   Z
Inverts the current selection (selected → unselected, unselected → selected).

Select by Filter

Edit › Select by Filter   W
Selects files that match a specified name pattern or condition.

Selection Style

TableEngine supports two selection styles, switchable from Edit › Selection Style.

Explorer Style

Same selection behavior as Windows Explorer.

  • Click to select a single item (deselects others)
  • Ctrl+click to add or remove from selection
  • Shift+click for range selection

TableEngine Style

A unique selection style inspired by FD / Takuchu☆.

  • Press Space to toggle selection on the cursor item and advance the cursor
  • Designed for efficient multi-file selection using only the keyboard
The TableEngine style is more efficient when selecting a large number of files using only the keyboard.
