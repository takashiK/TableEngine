第5章 選択・編集

ファイルの選択操作とクリップボード操作について説明します。 これらのコマンドは 編集 メニューからアクセスできます。

クリップボード操作

切り取り（Cut）

編集 › 切り取り   Ctrl+X
選択中のファイル・フォルダをクリップボードに切り取ります。 貼り付け（Paste）を実行すると元の場所から削除されます。

コピー（Copy）

編集 › コピー   Ctrl+C
選択中のファイル・フォルダをクリップボードにコピーします。

貼り付け（Paste）

編集 › 貼り付け   Ctrl+V
クリップボードのファイル・フォルダを現在のフォルダに貼り付けます。 Windows エクスプローラーとクリップボードを共有しています。

選択操作

すべて選択

編集 › すべて選択   Ctrl+A
現在のフォルダのすべてのファイル・フォルダを選択します。

選択の反転

編集 › 選択の反転
現在の選択状態を反転します（選択中 → 非選択、非選択 → 選択中）。

フィルタで選択

編集 › フィルタで選択
ファイル名のパターンや条件を指定して、一致するファイルを選択します。

→ ダイアログの詳細: フィルタダイアログ
オプション説明
フィルタワイルドカードまたは正規表現でファイル名を指定
ファイルのみフォルダを除外してファイルのみを対象にする
大文字小文字を区別大文字・小文字を区別して照合する
正規表現フィルタを正規表現として扱う

選択スタイル

TableEngine は 2 種類の選択スタイルをサポートしています。 編集 › 選択スタイル から切り替えられます。

エクスプローラースタイル

Windows エクスプローラーと同様の選択動作です。

  • クリックで単一選択（他の選択解除）
  • Ctrl+クリックで複数選択の追加・解除
  • Shift+クリックで範囲選択

TableEngine スタイル

FD / 卓駆☆ のような独自の選択動作です。

  • スペースキーでカーソル位置のファイルを選択・解除してカーソルを次に移動
  • 複数ファイルの連続選択がキーボードのみで行いやすい設計
大量のファイルをキーボードだけで選択する場合は TableEngine スタイルが効率的です。
