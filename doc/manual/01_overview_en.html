Chapter 1: Overview

What is TableEngine?

TableEngine is an open-source file manager for Windows. It was inspired by the keyboard-driven DOS file manager FD and its Windows successor Takuchu☆, and was created because neither of those applications supports Windows 10/11.

TableEngine carries on the keyboard-centric workflow of those classic filers while running comfortably on modern Windows.

Key Features

  • Dual-pane layout (folder tree + file list)
  • Tab support for managing multiple folders simultaneously
  • File compression and extraction (via libarchive)
  • File split and combine
  • Built-in text and binary viewers
  • Bulk file rename with sequential numbering
  • File search (by name, date, and size)
  • Customizable keyboard shortcuts
  • Customizable menu layout
  • Favorite folder bookmarks
  • Multiple selection styles (Explorer-style / TableEngine-style)
