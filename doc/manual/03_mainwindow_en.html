Area Descriptions

Menu Bar

Provides access to all features via menus: File / Edit / Folder / View / Tools / Window / Settings / Help. The menu layout can be customized in Settings.

Toolbar

Quick-access buttons for frequently used commands. Toggle visibility with Window › Show Toolbar.

Drive Bar

Displays available drives (C:, D:, etc.) as buttons. Clicking a drive button navigates to its root folder. Toggle with Window › Show Drive Bar.

Tab Bar

Manage multiple folders simultaneously with tabs. Press Ctrl+N to open a new tab and Ctrl+W to close the current tab. Tabs can also be moved to a separate window.

Navigation Pane

Shows a folder tree. Click a folder to navigate to it, updating the file list. Toggle with Window › Show Navigation.

File List

The main area showing files and folders in the current directory. The display mode can be changed between small icons, large icons, huge icons, or detail list. All file operations (select, copy, delete, etc.) are performed here.

Detail Pane

Shows detailed information (name, size, modification date, attributes, etc.) for the selected file. Toggle with Window › Show Details.

Status Bar

Displays item count, selection count, and total size for the current folder. Toggle with Window › Show Status Bar.
