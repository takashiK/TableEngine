第11章 ダイアログ一覧

TableEngine で使用される各ダイアログの操作方法を説明します。

パス入力ダイアログ

コピー・移動先などのパスを入力するダイアログです。

操作説明
パス入力欄宛先のパスを直接入力します。パス補完が有効です
Shift+Enterフォルダツリー選択ダイアログを開きます
↑移動履歴の一覧を表示します
↓お気に入りフォルダの一覧を表示します

フォルダ選択ダイアログ

フォルダツリーとパス入力欄を使ってフォルダを選択するダイアログです。 パス入力ダイアログで Shift+Enter を押すと表示されます。

  • ツリービューでフォルダをクリックすると、入力欄にパスが自動入力されます
  • 入力欄に直接パスを入力するとツリーがスクロールして該当フォルダを表示します
  • 入力欄でパス補完が使えます

検索ダイアログ（Find）

ファイルを検索するダイアログです。 フォルダ › 検索 で表示されます。

項目説明
検索ボックスファイル名のパターンを入力（例: *.txt）
Advanced ボタン詳細条件パネルを展開・折りたたみます
ファイル名 / ワイルドカード*・? を使ったパターンマッチ
ファイル名 / 正規表現正規表現によるパターンマッチ
大文字小文字を区別大文字・小文字を区別してマッチ
日付 / 種別なし・更新日時・作成日時から選択
日付 / 期間カレンダーポップアップで開始日〜終了日を指定
サイズ / 値比較するファイルサイズ（0〜999999）
サイズ / 単位KB または MB
サイズ / 条件「以上」または「以下」

フィルタダイアログ

ファイルリストの表示を絞り込むダイアログです。 表示 › フィルタ で表示されます。

項目説明
フィルタファイル名のパターンを入力
ファイルのみフォルダを除外してファイルのみを表示
大文字小文字を区別大文字・小文字を区別してフィルタリング
正規表現正規表現として入力を解釈

一括リネームダイアログ

複数ファイルを連番付きの名前に一括変更するダイアログです。 ファイル › 一括リネーム で表示されます。

項目説明
サンプル設定に基づいた変更後のファイル名プレビュー（読み取り専用）
ベース名ファイル名の基本文字列
拡張子変更後の拡張子（ドットなし、省略可）
桁数連番の桁数（1〜4）
開始番号連番の開始値（0〜9999）
順序 / ファイル名順ファイル名のアルファベット順で番号付け
順序 / 更新日時順更新日時の古い順で番号付け

ファイル分割ダイアログ

ファイルを指定サイズで分割するダイアログです。 ファイル › 分割 › ファイルを分割 で表示されます。

項目説明
ファイルパス分割対象のファイル名（読み取り専用）
出力先分割ファイルの保存先フォルダ
分割サイズ1 ファイルあたりのサイズ（1〜1024 MB）
分割数自動計算された分割数（読み取り専用）

ファイル結合ダイアログ

分割されたファイルを結合するダイアログです。 ファイル › 分割 › ファイルを結合 で表示されます。

項目説明
出力ファイル結合後のファイル名（相対パスまたは絶対パス）
... ボタンファイル保存ダイアログで出力先を選択
拡張選択同一ベース名の関連ファイル（.000、.001…）を自動追加
入力ファイル一覧結合対象のファイル一覧（ファイル名順）
結合サイズ結合後の合計サイズ（自動計算、読み取り専用）

コマンド実行ダイアログ

任意のコマンドラインを実行するダイアログです。 ファイル › コマンドを実行 で表示されます。

項目説明
コマンド入力欄実行するコマンドライン文字列
Shellチェック時は cmd.exe /c 経由で実行します
出力を表示チェック時はコマンドの標準出力をウィンドウに表示します

パスリストダイアログ

お気に入りなどのパスリストを編集するダイアログです。

  • 追加 ボタン（フォルダアイコン）: リストにパスを追加します
  • 削除 ボタン（ゴミ箱アイコン）: 選択中のパスを削除します
  • 上へ ボタン: 選択中のパスを上に移動します
  • 下へ ボタン: 選択中のパスを下に移動します
  • 右クリックで「追加」「削除」のコンテキストメニューも使用できます

フォント設定ダイアログ

フォント・文字色・背景色を設定するダイアログです。

項目説明
フォントフォントファミリーを選択します
サイズフォントのポイントサイズ（1〜200）
文字色クリックするとカラーピッカーが開きます
背景色クリックするとカラーピッカーが開きます
サンプル設定を反映したプレビューが表示されます
