Chapter 8: Tools

TableEngine includes built-in viewers for inspecting file contents. These are accessible from the Tools menu.

Text Viewer

Tools › View File   V
Opens the selected file in the built-in text viewer. Supports text files, HTML, Markdown, and other markup formats.

  • Syntax highlighting for source code files
  • Automatic character encoding detection
  • Toggle word wrap
  • Rendered display for Markdown and other markup formats
Select a file in the list and press V (or use Tools › View File) to open it quickly.

Binary Viewer

Tools › View Binary   B
Opens the selected file in a hex dump viewer. Use this for binary files or files that cannot be displayed correctly in the text viewer.

  • Side-by-side hexadecimal and ASCII display
  • Offset (address) column
Both viewers are read-only. File contents cannot be modified.
