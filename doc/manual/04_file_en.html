Chapter 4: File Operations

File operation commands are accessible from the File menu. You can also right-click in the file list to use the context menu.

New File

File › New File   N
Creates an empty file in the current folder. Enter the file name in the dialog that appears.

Open

File › Open
Opens the selected file with the associated Windows shell application.

Double-clicking a file performs the same action.

Copy To

File › Copy To   C
Copies the selected files/folders to a specified destination folder. A path input dialog will appear.

  • Type the destination path directly, or select a folder from the tree view.
  • Press Shift+Enter to open the folder tree selection dialog.
  • Press ↑ to show history, ↓ to show favorites.
→ Dialog details: Path Input Dialog, Folder Select Dialog

Move To

File › Move To   M
Moves the selected files/folders to a specified destination folder. The operation is the same as Copy To.

If the destination does not exist, you will be asked whether to create a new folder or rename the file on copy.
→ Dialog details: Path Input Dialog, Folder Select Dialog

Delete

File › Delete   D / Delete
Deletes the selected files/folders. A confirmation dialog will appear.

Deleted files are not sent to the Recycle Bin. Please verify before proceeding.

Rename

File › Rename   R / F2
Renames the selected file.

Bulk Rename

File › Bulk Rename
Renames multiple selected files at once using a sequential number pattern.

→ Dialog details: Bulk Rename Dialog
FieldDescription
Base NameThe base string for the new file names
ExtensionNew file extension (optional)
DigitsNumber of digits in the sequential number (1–4)
Start NumberFirst value in the sequence
OrderBy file name / by modification date

Example: Base=photo, Digits=3, Start=1 → photo001.jpg, photo002.jpg…

Split File

File › Split › Split File
Splits a single file into multiple parts of the specified size.

→ Dialog details: File Split Dialog
FieldDescription
Split SizeSize per output file (in MB)
Output FolderDestination folder for the split files
Part CountCalculated automatically (read-only)

Combine Files

File › Split › Combine Files
Combines previously split files (e.g., file.000, file.001…) into a single file.

  • Enabling "Extended Selection" automatically finds related files that share the same base name.
  • Specify the output file name to complete the combine.
→ Dialog details: File Combine Dialog

Archive (Compress)

File › Archive › Compress   P
Compresses the selected files/folders into an archive. Supported formats depend on libarchive (zip, tar.gz, 7z, etc.).

Extract

File › Archive › Extract   U
Extracts the selected archive file. You can specify the destination folder.

Properties

File › Properties   F4
Displays detailed information (size, date, attributes, MIME type, etc.) for the selected file.

Run Command

File › Run Command   X
Executes an arbitrary command line.

→ Dialog details: Run Command Dialog
OptionDescription
Run via ShellExecutes through the shell (cmd.exe)
Show OutputDisplays the command's standard output in a window

Exit

File › Exit   Escape
Exits TableEngine.
