Chapter 6: Folder Operations

This chapter covers folder creation, tree expansion, navigation, favorites, and search. These commands are accessible from the Folder menu.

New Folder

Folder › New Folder
Creates a new folder inside the current folder. Enter the folder name in the dialog that appears.

Expanding and Collapsing the Folder Tree

Commands for controlling the folder tree in the navigation pane.

CommandMenuDescription
Expand One Level Folder › Expand One Level Expands the selected folder by one level
Expand All Below Folder › Expand All Below Expands the selected folder and all its descendants
Expand All Folder › Expand All Expands the entire tree
Collapse Folder › Collapse Collapses the selected folder
Collapse All Folder › Collapse All Collapses the entire tree

Folder Navigation

Back / Forward

Folder › Back   Alt+←
Folder › Forward   Alt+→
Navigate through folder history, similar to a browser's Back and Forward buttons.

Go to Parent Folder

Folder › Go to Parent   Backspace
Moves up to the parent folder.

Go to Root Folder

Folder › Go to Root   Home
Navigates to the root of the current drive (e.g., C:\).

Go to Folder

Folder › Go to Folder   G
Opens a path input dialog to navigate to any folder by typing its path. Press ↑ for history or ↓ for favorites in the input field.

→ Dialog details: Path Input Dialog

Change Root Folder

Folder › Change Root Folder
Changes the root of the navigation pane tree to a different folder. Useful for limiting the display to a specific subtree.

Favorites

Add to Favorites

Folder › Add to Favorites
Adds the current folder to the favorites list.

Edit Favorites

Folder › Edit Favorites
Manages the favorites list. You can add, delete, and reorder entries.

→ Dialog details: Path List Dialog
In the Go to Folder input field, press ↓ to display the favorites list for quick navigation.

Search

Folder › Search   F / Ctrl+F
Searches for files under the current folder.

→ Dialog details: Find Dialog

Basic Search

Enter a file name pattern (wildcards supported) in the search box.

  • Example: *.txt → find all text files
  • Example: report* → find files starting with “report”

Advanced Search

Click the “Advanced” button to expand the detailed search panel.

ConditionDescription
Name › WildcardMatch using * and ? patterns
Name › Regular ExpressionMatch using a regular expression
Name › Case SensitiveDistinguish upper and lower case
Date › TypeNone / Modified / Created
Date › RangeStart date to end date
SizeGreater than or less than a specified size (KB or MB)
