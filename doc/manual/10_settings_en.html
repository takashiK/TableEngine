Chapter 10: Settings

This chapter covers how to customize TableEngine's behavior. Each settings dialog is opened from the Settings menu.

Options

Settings › Options
General application settings.

Examples of configurable items:

  • Startup folder
  • Font and color settings (via the font selection dialog)
  • Favorites list management — see: Path List Dialog
  • History entry limit

Font & Color Settings

The font settings dialog lets you change the following.

→ Dialog details: Font Settings Dialog
ItemDescription
FontFont family used for display
SizeFont point size (1–200)
Text ColorForeground color of the text
Background ColorBackground color behind the text

Key Settings

Settings › Key Settings
Customize the keyboard shortcuts assigned to commands.

  • Each command can have up to two shortcuts assigned
  • Existing assignments can be removed and replaced with custom keys
  • Includes a reset-to-defaults function
Key settings are saved to the configuration file and persist across application updates.

Menu Settings

Settings › Menu Settings
Customize the command layout in the menu bar and context menus.

  • Change the order of commands
  • Hide commands you do not use
  • Create user-defined command groups
Placing frequently used commands near the top of menus speeds up your workflow.
