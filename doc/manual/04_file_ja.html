第4章 ファイル操作

ファイル操作コマンドは ファイル メニューから実行できます。 また、ファイルリストで右クリックしてコンテキストメニューから実行することもできます。

新規作成

ファイル › 新規作成
空のファイルを現在のフォルダに作成します。 ダイアログでファイル名を入力してください。

ファイルを開く

ファイル › 開く
選択中のファイルを、Windows のシェル（関連付けられたアプリケーション）で開きます。

ファイルをダブルクリックしても同じ動作をします。

コピー先にコピー

ファイル › コピー先にコピー
選択中のファイル・フォルダを、指定したフォルダにコピーします。 コピー先を指定するダイアログが表示されます。

  • パス入力欄に直接パスを入力するか、ツリービューからフォルダを選択します。
  • Shift+Enter でフォルダツリー選択ダイアログを表示します。
  • ↑ キーで履歴、↓ キーでお気に入りを表示します。
→ ダイアログの詳細: パス入力ダイアログ、フォルダ選択ダイアログ

移動先に移動

ファイル › 移動先に移動
選択中のファイル・フォルダを、指定したフォルダに移動します。 操作方法はコピーと同様です。

移動先が存在しない場合、新規フォルダを作成するか、ファイル名を変更してコピーするかを選択するダイアログが表示されます。
→ ダイアログの詳細: パス入力ダイアログ、フォルダ選択ダイアログ

削除

ファイル › 削除
選択中のファイル・フォルダを削除します。確認ダイアログが表示されます。

削除したファイルはごみ箱には入りません。操作前に十分ご確認ください。

名前の変更

ファイル › 名前の変更
選択中のファイルの名前を変更します。

一括リネーム

ファイル › 一括リネーム
選択中の複数ファイルを、連番付きの名前に一括変更します。

→ ダイアログの詳細: 一括リネームダイアログ
項目説明
ベース名ファイル名の基本となる文字列
拡張子変更後の拡張子（省略可）
桁数連番の桁数（1〜4桁）
開始番号連番の開始値
順序ファイル名順 / 更新日時順

例: ベース名=photo、桁数=3、開始=1 → photo001.jpg、photo002.jpg…

ファイルの分割

ファイル › 分割 › ファイルを分割
1つのファイルを指定したサイズで複数のファイルに分割します。

→ ダイアログの詳細: ファイル分割ダイアログ
項目説明
分割サイズ1ファイルあたりのサイズ（MB単位）
出力先分割ファイルの保存先フォルダ
分割数自動計算して表示（読み取り専用）

ファイルの結合

ファイル › 分割 › ファイルを結合
分割されたファイル（例: file.000、file.001…）を1つのファイルに結合します。

  • 「拡張選択」を有効にすると、選択ファイルと同じベース名を持つ関連ファイルを自動検索して追加します。
  • 出力ファイル名を指定して結合します。
→ ダイアログの詳細: ファイル結合ダイアログ

圧縮

ファイル › アーカイブ › ファイルを圧縮
選択中のファイル・フォルダを圧縮ファイルにまとめます。 対応形式は libarchive のサポートに準じます（zip、tar.gz、7z など）。

展開

ファイル › アーカイブ › ファイルを展開
選択中の圧縮ファイルを展開します。展開先フォルダを指定できます。

プロパティ

ファイル › プロパティ
選択中のファイルの詳細情報（サイズ・日時・属性・MIMEタイプなど）を表示します。

コマンドを実行

ファイル › コマンドを実行
任意のコマンドラインを実行します。

→ ダイアログの詳細: コマンド実行ダイアログ
オプション説明
Shell で実行シェル（cmd.exe）経由で実行します
出力を表示コマンドの標準出力をウィンドウに表示します

終了

ファイル › 終了
TableEngine を終了します。
