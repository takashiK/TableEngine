第2章 インストールと起動

インストール手順

TableEngine にはインストーラーはありません。配布パッケージ（ZIP ファイル）を展開するだけでインストール完了です。

  1. 配布パッケージ（例: TableEngine-0.1.0-win64.zip）をダウンロードします。
  2. ZIP ファイルを任意のフォルダ（例: C:\TableEngine）に展開します。
  3. 展開後、フォルダ内の TableEngine.exe を実行するとアプリが起動します。

アンインストール

TableEngine はレジストリを使用しません。 設定ファイルはユーザーのアプリデータフォルダに保存されます。
完全に削除するには、インストールフォルダと以下の設定フォルダを削除してください。 

%APPDATA%\TableEngine\

起動

TableEngine.exe をダブルクリックするか、コマンドラインから実行します。 

TableEngine.exe
スタートメニューやタスクバーにピン留めして使うと便利です。

ビルド方法（開発者向け）

ソースコードからビルドするには以下の手順を実行してください。

前提条件

  • Qt 6.11 以降（MSVC 2022 64-bit キット。Ninja もここに含まれる）
  • CMake 3.19 以降
  • Microsoft Visual C++ 2022
  • Git（サブモジュールの取得に必要）

ビルド手順

1. リポジトリのクローンとサブモジュールの初期化:

git clone <repo-url> TableEngine
cd TableEngine
git submodule update --init --recursive

2. vcpkg のブートストラップ（初回のみ）:

support\vcpkg\bootstrap-vcpkg.bat -disableMetrics

3. CMake プリセットで構成・ビルド:

cmake --preset Qt-MSVC2022-amd64-Ninja
cmake --build out/build/Qt-MSVC2022-amd64-Ninja --config Release

4. 配布パッケージの作成:

.\scripts\package-release.ps1
