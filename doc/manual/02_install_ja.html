第2章 インストールと起動

必要なファイル

TableEngine を実行するには、以下のファイルが TableEngine.exe と同じフォルダに 必要です。

ファイル名説明
TableEngine.exe本体実行ファイル
archive.dlllibarchive（圧縮・展開ライブラリ）
libmagic.dllファイル種別判定ライブラリ
magic.mgclibmagic データベース
Qt DLL 群Qt6Core.dll、Qt6Gui.dll、Qt6Widgets.dll など
Qt DLL は Qt のインストールフォルダまたは windeployqt ツールを使って 収集することができます。

インストール手順

TableEngine にはインストーラーはありません。以下の手順でフォルダに配置してください。

  1. 任意のフォルダ（例: C:\Tools\TableEngine\）を作成します。
  2. ビルド済みの TableEngine.exe をそのフォルダにコピーします。
  3. 依存 DLL（archive.dll、libmagic.dll、magic.mgc）を同フォルダにコピーします。
  4. Qt DLL 群を同フォルダにコピーします。
    コマンドプロンプトで以下を実行すると自動収集できます: 
    windeployqt.exe TableEngine.exe

アンインストール

TableEngine はレジストリを使用しません。 設定ファイルはユーザーのアプリデータフォルダに保存されます。
完全に削除するには、インストールフォルダと以下の設定フォルダを削除してください。 

%APPDATA%\TableEngine\

起動

TableEngine.exe をダブルクリックするか、コマンドラインから実行します。 

TableEngine.exe
スタートメニューやタスクバーにピン留めして使うと便利です。

ビルド方法（開発者向け）

ソースコードからビルドするには以下の手順を実行してください。

前提条件

  • Qt 6.10 以降（Qt6_DIR または CMAKE_PREFIX_PATH で指定）
  • CMake 3.19 以降
  • Microsoft Visual C++ 2022
  • サポートライブラリ（libarchive、libmagic、valijson）を support_package/ に配置

ビルド手順

cmake --preset "Visual Studio Community 2022 Release - amd64"
cmake --build out/build/"Visual Studio Community 2022 Release - amd64" --config Release

または Visual Studio で開く場合:

cmake --preset "Visual Studio Community 2022 Release - amd64"
start out/build/"Visual Studio Community 2022 Release - amd64"/TableEngine.sln
