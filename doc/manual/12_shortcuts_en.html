Chapter 12: Keyboard Shortcuts

Default keyboard shortcut reference for TableEngine. Shortcuts can be customized via Settings › Key Settings.

The shortcuts below are the defaults (defined in TeKeySetting::storeDefaultSettings). They may differ if you have customized your key settings.

File Operations (No Modifier)

KeyCommandDescription
NNew FileCreate an empty file in the current folder
CCopy ToCopy selected files to a specified folder
MMove ToMove selected files to a specified folder
D / DeleteDeleteDelete selected files
R / F2RenameRename the selected file
F4PropertiesShow properties of the selected file
PArchive (Compress)Compress selected files into an archive
UExtractExtract a compressed archive
XRun CommandOpen the Run Command dialog
EscapeExitExit TableEngine

Selection & Editing (No Modifier)

KeyCommandDescription
ASelect AllSelect all files in the current folder
ZInvert SelectionInvert the current selection
WSelect by FilterSelect files matching a filter condition

Selection & Editing (Ctrl Key)

KeyCommandDescription
Ctrl+CCopy (Clipboard)Copy selected files to the clipboard
Ctrl+VPastePaste clipboard files into the current folder
Ctrl+XCutCut selected files to the clipboard
Ctrl+ASelect AllSelect all files in the current folder

Folder Navigation (No Modifier)

KeyCommandDescription
BackspaceGo to Parent FolderMove up to the parent folder
HomeGo to Root FolderNavigate to the drive root (e.g., C:\)
GGo to FolderOpen the path input dialog
FSearchOpen the search dialog

Folder Navigation (Ctrl Key)

KeyCommandDescription
Ctrl+NNew FolderCreate a new folder in the current location
Ctrl+FSearchOpen the search dialog

View (No Modifier)

KeyCommandDescription
OShow Hidden FilesToggle hidden file visibility
YFilterFilter files by name

View (Ctrl Key)

KeyCommandDescription
Ctrl+IHuge Icon ViewSwitch file list to huge icon layout
Ctrl+LSmall Icon ViewSwitch file list to small icon layout

Tools (No Modifier)

KeyCommandDescription
VView FileOpen the selected file in the text viewer
BView BinaryOpen the selected file in the binary viewer

Window (Ctrl Key)

KeyCommandDescription
Ctrl+TNew TabOpen a new tab
Ctrl+WClose TabClose the current tab
Ctrl+MMove Tab to New WindowDetach the current tab into a new window
Ctrl+JNext TabSwitch to the next tab
Ctrl+KPrevious TabSwitch to the previous tab

Navigation (No Modifier)

KeyCommandDescription
JCursor DownMove cursor down
KCursor UpMove cursor up
HCursor LeftMove cursor left
LCursor RightMove cursor right
QNavigate DriveBarMove focus to the drive bar
TabToggle Folder Left/RightToggle focus between left and right folder pane

Navigation (Ctrl Key)

KeyCommandDescription
Ctrl+TabToggle Folder/TreeToggle focus between folder pane and tree view
Shortcuts can be freely customized from Settings › Key Settings. Assign your most-used operations to convenient keys to maximize efficiency.
